課程名稱：無線通訊專論 (Special topics on wireless communications)

授課老師：韓永祥

課程目標：本課程之目標是教導學生無線通訊中安全機制的知識。

成績考查方式：期中專題 40%，期末專題50％，平常成績佔10％

A set of lecture notes will be provided on class. The following are the textbooks:

1. Security in Fixed and Wireless Networks, G. Schafer, John Wiley & Sons, Inc., 2003. (全華科技書局代理)

2. Network Security Essentials, 2nd Edition, W. Stallings, Prentice Hall, 2003.

 

.

 

Students’final grades for this course are based on their performances in midterm and final project reports on problems in the security research area, and up to 120-minute oral presentations on their reports. Students will be asked to attend every lecture in order to learn the advance topics of security on wireless communications.

 

課程綱要：

1.          Cryptographic protocol

2.          Security aspects of mobile communication

3.          Security in wireless local area networks

4.          Security in mobile wide-area networks
